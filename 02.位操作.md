#02.位操作

##2.1 按位与

逐个位元进行条件与操作。

```
  0100 1001
& 0101 0011
-----------
  0100 0001
```

##2.2 按位或

逐个位元进行条件或操作。


```
  0100 1001
| 0101 0011
-----------
  0101 1011
```

##2.3 按位异或

两个字组，对应相同的位元取值1，其他则为0。

```
  0100 1001
^ 0101 0011
-----------
  1110 0101
```

##2.4 左移

```
  0100 1001
<<4 
-----------
  1001 0000
```

##2.5 右移

```
  0100 1001
>>2 
-----------
  0001 0010
```
